Usage:
- To enable the visual simulation, an additional numeric argument should be passed to the program indicating the desired skin:
  - 0 = Regular
  - 1 = Tennis
  - 2 = Aliens
  - 3 = Pirate
  - 4 = Middle East
- If no number is given, the game runs without visual simulation by default.
